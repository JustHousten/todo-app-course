# Todo-App-Kurs

Diese Repo ist ein **Lernkurs**, kein fertiges Projekt. Ziel: JavaScript und danach
Node.js wirklich verstehen, indem eine eigene Todo-App komplett selbst gebaut wird –
Schritt für Schritt, Lektion für Lektion. Ein endgültiger Name für die App steht noch
nicht fest, im Code heißt sie vorerst einfach `todo-app`.

Es gab eine ältere Todo-App (Cloudflare-Hosting, `localStorage`, damals "Fokus"
genannt). Die wird hier **nicht wiederverwendet** – sie dient nur als Vorbild dafür,
wie sich die neue App anfühlen und was sie können soll.

## Wie diese Repo aufgebaut ist

- **`course/`** – die eigentlichen Lektionen als Markdown-Dateien, in Reihenfolge
  durchnummeriert. Jede Lektion hat einen `exercises/`-Unterordner mit
  Starter-Dateien zum Selbst-Ausfüllen.
- **`project/`** – hier wächst die eigentliche Todo-App im Kursverlauf. Am Anfang
  leer, Stück für Stück kommt hier echter Code rein.
- **`PROGRESS.md`** – laufendes Log: was schon gemacht wurde, wo wir gerade stehen,
  offene Fragen. Hier reinschauen, wenn nach einer Pause der Wiedereinstieg ansteht.
- **`course/00-roadmap.md`** – der komplette Kursplan im Überblick (alle Phasen).

Alle Dokumentation (READMEs, Lektionen, Fahrplan, Fortschritts-Log) existiert in
zwei Versionen: Englisch (Standard) und Deutsch (`*.de.md`) – gleicher Inhalt,
falls es mal jemanden interessiert, wie dieser Weg gegangen wurde. Code- und
Übungsdateien bleiben, wie in der Branche üblich, ausschließlich Englisch.
