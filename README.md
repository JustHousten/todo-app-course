# Todo App Course

This repo is a **learning course**, not a finished project. Goal: really understand
JavaScript, and later Node.js, by building an own todo app completely from scratch —
step by step, lesson by lesson. A final name for the app is not decided yet; in code
it's just called `todo-app` for now.

There was an older todo app (Cloudflare hosting, `localStorage`, back then called
"Fokus"). It is **not reused** here — it only serves as a reference for how the new
app should feel and what it should be able to do.

## How this repo is structured

- **`course/`** — the actual lessons as markdown files, numbered in order. Each
  lesson has an `exercises/` subfolder with starter files to fill in yourself.
- **`project/`** — this is where the actual todo app grows over the course. Empty
  at first, real code arrives piece by piece.
- **`PROGRESS.md`** — ongoing log: what's been done, where things currently stand,
  open questions. Check here when picking this back up after a break.
- **`course/00-roadmap.md`** — the full course plan at a glance (all phases).

All documentation (READMEs, lessons, roadmap, progress log) exists in two
versions: English (default) and German (`*.de.md`) — same content, in case anyone
else is curious how this journey went. Code and exercise files stay English-only,
following common convention.
